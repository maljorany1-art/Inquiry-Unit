
5. توثيق الشغل على GitHub
عشان أحفظ الكود وما يضيع وأوثق كل خطوة عملتها، استخدمت موقع GitHub، وهذه هي الخلاصة:
أولاً: عملت المستودع (Repository):
فتحت مشروع جديد على حسابي في GitHub وسميته Inquiry-Unit، وهذا هو المكان اللي رفعت عليه كل الأكواد والملفات.
رابط المشروع: https://github.com/maljorany1-art/Inquiry-Unit
ثانياً: خطوات رفع الشغل:
استخدمت برنامج NetBeans نفسه عشان أرفع الملفات، وعملت 3 خطوات:
Add: جهزت ملفات الجافا والمشروع للرفع.
Commit: كتبت وصف بسيط للشغل اللي رفعته عشان أعرف التحديثات اللي عملتها.
Push: وهذه كانت أهم خطوة، وهي إرسال الملفات من جهازي للموقع مباشرة باستخدام Token الأمان
